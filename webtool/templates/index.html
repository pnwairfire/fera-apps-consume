Consume Webtool

Run the two-step fuel consumption & emissions pipeline for selected FCCS fuelbeds.

1. Configure Run

Each fuelbed automatically generates 27 disturbance variations (codes 111–333) and 7 fire scenarios, for up to 28 × 7 = 196 run-1 rows per fuelbed. Run 2 then produces 7 additional scenario combinations per run-1 row.
▸ Advanced options
Default: consume/input_data/fccs_loadings.csv from this repository.
Each scenario becomes one fire condition applied to every fuelbed variation. Up to 9 scenarios supported.
